# 荔枝小秘书（Lychee Assistant）

[English](./README.en.md) | 简体中文

> 一款用于辅助 AI 编程的工具网站，项目名称：`lychee`。

## 项目简介

我想开发一个 AI 编程辅助工具网站，名字叫**荔枝小秘书**（英文名 **Lychee Assistant**），用来辅助 AI 编程。

## 技术栈

- **后端框架**：Gin
- **ORM**：GORM
- **数据库**：SQLite
- **模板引擎**：Go 标准 `html/template` 及简易 Jet 模板（用于渲染前端页面）

## 功能特性

- **多语言支持**：暂时支持英语和中文，可通过页面上的按钮进行切换（默认使用系统语言）。
- **注册功能**：支持通过邮箱注册，注册时发送邮箱验证码进行验证。
- **登录功能**：使用邮箱和密码登录，采用 JWT 认证；每个用户的数据相互隔离，并支持双 Token 无感刷新。
- **忘记密码**：可通过邮箱验证码重置密码。
- **个人设置**：可通过邮箱验证码修改密码。
- **常用语管理**：每位用户拥有一套自己的常用语，可在常用语管理列表页面进行增删改查。
- **任务列表**：用于管理任务提示词，主要分为左右两栏：
  - **左侧 — 分组管理**：支持分组，且分组内可以嵌套分组。
  - **右侧 — 任务管理**：点击左侧任意分组后，右侧展示该分组的任务列表。
    - 任务展示字段：任务内容、创建时间、修改时间、截止日期、优先级（P0–P5）、状态（编辑中、待执行、已完成）。
    - 支持按创建时间、修改时间、截止日期、优先级进行排序。
    - 操作按钮：
      - **编辑**：弹框修改任务内容、优先级、状态；编辑时可选择常用语，将其填充至任务内容输入框的光标位置。
      - **复制**：将任务内容复制到剪贴板。
- **打卡功能**：可记录每次喝水（饮水量）、起身运动（时长，分钟）、午睡（时长，分钟）等，支持查看每日打卡报告，并可自定义健康打卡的每日目标。
  - 后续将增加 **AI 健康分析**功能，根据打卡数据自动分析健康状况并给出建议。
- **操作日志**：记录操作日志（包括登录日志），可查看自己的操作历史，以及操作报告（每天、每小时的操作次数，以柱状图展示）。

## 注意事项

- **接口限流**：同一个接口，每个 IP 每秒只能访问一次。
