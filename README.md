我想开发一个AI编程辅助工具网站，名字叫荔枝小秘书，英文名 Lychee Assistant，用来辅助AI编程，项目名称：lychee。
项目技术栈使用：Gin + GORM + SQLite，Go 标准 html/template 及简易 Jet 模板（渲染前端页面）。
主要功能如下：
●多语言功能，暂时支持英语和中文，可以通过页面上的按钮进行切换（默认用系统语言）；
●注册功能，可以通过邮箱注册（发送邮箱验证码进行验证）；
●登录功能，使用邮箱和密码登录，采用 JWT 认证，每个用户数据相互隔离（双 Token 无感刷新）；
●忘记密码功能，可以通过邮箱验证码修改密码；
●个人设置功能，可以通过邮箱验证码修改密码；
●常用语管理，每个人有一套自己的常用语，可以在常用语管理列表页面进行增删改查；
●任务列表功能，可以用来管理任务提示词。主要分为左右两栏，左边一栏是分组管理，分组里面可以嵌套分组，右边一栏是任务管理，点击左边的任意一组，右边就展示左边分组的任务列表。任务列表中，展示任务内容、创建时间、修改时间、截止日期、优先级（P0-P5）、状态（编辑中、待执行、已完成），可以按创建时间、修改时间、截止日期、优先级排序，最右边有操作按钮：编辑（可以弹框修改任务内容、优先级、状态，编辑时可选择常用语填充至任务内容输入框光标位置），复制（将任务内容复制到剪贴板）；
●打卡功能，每次喝水（饮水量）、起身运动（时长，分钟）、午睡（时长，分钟）等，都可以记录下来，可以查看每天的打卡报告，可以自定义健康打卡每日目标，后续将增加 AI 健康分析功能，根据打卡数据自动分析健康状况并给出建议；
●记录操作日志（包括登录日志），可以查看自己的操作历史，以及操作报告（每天、每小时的操作次数，以柱状图展示）；
注意事项：接口限流，同一个接口，每个 IP 每秒只能访问一次。
